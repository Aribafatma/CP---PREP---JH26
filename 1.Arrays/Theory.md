Arrays
Theory=Arrays store elements in contiguous memory locations.

Operations:

Access → O(1)
Search → O(n)
Insert/Delete → O(n)
Common Concepts
Traversal
Frequency Counting
Subarrays
Kadane's Algorithm
Prefix Sum

Important Patterns
Frequency Array=Count occurrences of elements.
Running Sum=Maintain cumulative values while traversing.
Kadane's Algorithm=Maximum Subarray Sum.
Range Queries=Use Prefix Sum.
Subarray Problems=Most array questions involve subarrays.
